Getting Started

The following aspects need to be configured before being able to use the plugin.
  • Installing the required Python libraries
  • Seting up for Notarisation
  • Seting up the Working Directory
  • Individual or Combined Mode

Installing the required Python libraries

Python should be installed on the machine # e.g. from windows - https://www.python.org/downloads/windows/ - select windows installer
# along with pip
# For example, in a Windows PC, run OSGeo4W.bat from the directory 'D:\QGIS' not from 'D\OSGEO4W'

The following libraries are required pip install in qgis
Use D:\QGIS\OSGEO.bat
D:\QGIS>pip install cryptography
D:\QGIS>pip install cryptography
D:\QGIS>pip install web3
D:\QGIS>pip install python-dotenv
D:\QGIS>pip install xkcdpass
D:\QGIS>pip install easygui or python -m pip install easygui
D:\QGIS>pip install python-dotenv
D:\QGIS>(python -m) pip install pyqt-switch

Seting up for Notarisation

All data should be notarised on the Ethereum mainnet. However, for testing one can use the public Ethereum testnet, such as Goerli. To mint data on ether networks, you need to carry out these tasks.

  • First, visit Vanity-ETH to generate an Ethereum vanity address and a private key.
  • Second, using your generated Ethereum address, request Ether from this Goerli Faucet
  • Third, install the metamask wallet, and connect it to the Goerli testnet or mainnet.
  • If using the testnet, you can check the Ether balance and all minted data on your address, at the Goerli etherscan

Seting up the Working Directory

It is essential for users to set the working directory. The plugin uses this diretory to Save the resulting masked, binned, and encrypted datasets. This directory is also used to perform computations.

This value can be set in the .env file or set by users when the plugin has been loaded.

Individual or Combined Mode

The plugin can be used in two modes. The individual mode allows users to use the plugin's three security functions (masking or hexabinning, encryption and notarisation) individually or in any combination.

On the other hand, the combined mode enforces and guides the user in perfoming these three functions one after the other in a workflow.

The switch at the top left permits this selection.
